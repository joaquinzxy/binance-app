# Pasitos pa dockerizar

- Mandarle cumbia al docker file 🐳

```
docker build -t binance-image .
```

- Ta pelao pa darle run (exponiendo los puertos nocierto?) 🏃‍♂️

```
docker run --name container-binance -p 3000:3000 binance-image
```
